Olá! Me chamo Nicollas e sou aspirante a desenvolvedor de software apaixonado por tecnologia, programação e aprendizado contínuo. Atualmente como estudante técnico de desenvolvimento de sistemas e aspirante a cientista da computação, direcionando meus estudos e projetos para o desenvolvimento full-stack.

No back-end possuo experiência com C# utilizando frameworks como .NET e ASP.NET Core, além de trabalhar com bancos de dados relacionais como SQL Server. No front-end, utilizo JavaScript e bibliotecas como GSAP para criar interfaces dinâmicas e interativas.

Durante meus estudos, venho desenvolvendo projetos práticos aplicando metodologias ágeis como Kanban e Scrum, além de conceitos de arquitetura de software como MVC e Domain-Driven Design (DDD). Também possuo experiência na criação de APIs RESTful utilizando C# e ASP.NET Core, sempre buscando boas práticas de desenvolvimento, organização de código e escalabilidade das aplicações.

Estou constantemente buscando aprimorar minhas habilidades, aprender novas tecnologias e contribuir em projetos que utilizem a tecnologia para resolver problemas reais.
